# knight-travails

This project implements a solution to the Knight's Travails problem using JavaScript. The goal is to find the shortest path a knight can take between two points on a chessboard.

## Getting Started

To run this project, you will need Node.js installed on your computer. After cloning the repository, you can run the script directly using Node.

### Prerequisites

- Node.js

### Running the Program

1. Open your terminal.
2. Navigate to the directory where you have the project files.
3. Run the script using Node
